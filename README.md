# Streamlit-Weather

## Исследование распараллеливания анализа

Решил сравнить анализ с помощью pandas и polars. Polars будет удобно распараллелить по городам. Получаем выигрыш от его использования в 3-4 раза даже на таком маленьком датасете.

## Исследование запросов

При одиночном запросе выиграша от использования асинхронного запроса нет. Если же сделать несколько запросов (например, я попробовал поискать, есть ли хоть в каком-то городе аномальная температура), то тогда выигрыш получается в среднем в 14 раз.
